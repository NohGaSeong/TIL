`서버: 요청(HTTP 요청)을 받으면 요청한 내용을 보내주는 프로그램`

## HTTP의 4가지 요청
1. 읽기
2. 쓰기
3. 수정
4. 삭제

### 읽기(GET)
- 나 이런 페이지 읽고싶어! 라는 요청 기능

### 쓰기(POST)
- 댓글 작성, 블로그 포스트 작성 등을 할때 쓰는 기능
- "이거 글 좀 올려주세용 ~ "

### 수정(PUT)
- 뭔가 수정할 때
### 삭제(DELETE)
- 뭔가 삭제할 때

최종 정리!
`서버: 요청(사용자는 GET/POST/PUT/DELETE 요청 가능)을 받으면 요청한 내용을 보내주는 프로그램`
